Icons

Icon sizing

Tiny

Tiny icons are 20px square. They were first used for the social media icon in the footer and can be applied using a ic-tiny class.

Small

Small icons are 40px square. First use, What's in the box? panel, home page. They can be applied using a ic-small class.

Medium

Medium icons are 60px square. First use, Lose 4.4 lbs in 10 days panel, home page. They can be applied using an ic-medium class.
